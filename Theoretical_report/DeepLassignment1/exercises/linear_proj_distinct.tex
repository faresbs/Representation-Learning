\Exercise{
\label{ex:linear_proj_distinct}
Given any unit vector $\vn$ (i.e. $||\vn||=1$), we define the hyperplane $\gH_\vn:=\{\vx:\vn^\top\vx=0\}$ for which $\vn$ is known as the normal vector. 
For any vector $\vx$, we define its projection into $\gH_\vn$ as $\pi_\vn(\vx)=\vx-(\vx^\top\vn)\vn$.

\begin{enumerate}
\item Given two vectors $\vx_1\neq\vx_2$, take $\vn=\frac{\vx_2-\vx_1}{||\vx_2-\vx_1||}$.
Show that $\pi_\vn(\vx_1)=\pi_\vn(\vx_2)$. 
\item Let $\vw$ be a vector and define $y_1:=\vx_1^\top\vw$ and $y_2:=\vx_2^\top\vw$. 
Show that $y_1=y_2$ if and only if $\vw\in\gH_\vn$. 
\staritem Let $\mX$ be a $n$ by $p$ matrix whose rows $\mX_{i, :}$ are all distinct.  
Show that there exists a vector $\vw$ of length $p$ such that the scalars $(\mX\vw)_i$ are all distinct. 
\end{enumerate}
